Analyze a 1D structural element (bar/rod) for internal forces, stresses, and strains.

Steps:
1. Define element properties: length L, cross-section area A, Young's modulus E
2. Define loading: axial forces, self-weight, temperature changes
3. Calculate:
   - Internal axial force N(x)
   - Normal stress σ = N/A
   - Normal strain ε = σ/E = ΔL/L
   - Elongation δ = NL/(AE) or δ = ∫(N/AE)dx
4. Plot N(x), σ(x), ε(x) diagrams
5. For composite bars: compatibility + equilibrium equations

Key formulas:
- Elongation: δ = PL/(AE)
- Stress: σ = P/A [Pa or MPa]
- Strain: ε = δ/L [dimensionless]
- Hooke's law: σ = E·ε
- Thermal strain: ε_T = α·ΔT
- Thermal stress (restrained): σ = E·α·ΔT

Sign convention:
- Positive N = tension
- Negative N = compression

Always include units: Pa or MPa for stress, m or mm for displacement.
