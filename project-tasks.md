# Course: Computer Applications in Civil Engineering (1090340)


- **Prerequisites:** Computer Programming for Civil Engineering (1022331)


---

## Goal
Provide the tools required to use Python for engineering calculations and solving numerical problems in engineering.

---

## Learning Outcomes
Upon successful completion, students will be able to:
1. Apply algebraic operators for vector calculus: sum of forces in a system, moments about a point, moments about an axis
2. Write Python programs to find internal forces, stresses, and strains in a 1D element
3. Write Python programs to find internal forces in statically determinate beams
4. Perform parametric studies for static schemes of beams and frames, and for a 1D element (elongation formula), using Python

---

## Weekly Schedule

| Week | Topic |
|------|-------|
| 1 | Python refresher, using AI for code writing |
| 2 | Vector calculus with numpy: sum of forces, moments about a point, moments about an axis |
| 3 | Internal force calculation in a 1D element |
| 4 | 1. Plotting · 2. Internal forces in statically determinate beams |

1. Svein Linge, Hans Petter Langtangen — *Programming for Computations: A Gentle Introduction to Numerical Simulations with Python* (2016)

## Teaching Method
- Fully online course — 6 sessions × 2 hours each
- Each session includes a Jupyter notebook with exercises
- Students attempt each task independently, then the lecturer solves it live
- 3 modules: Python refresher · basic numpy · numerical analysis with numpy for engineering problems
- **One homework assignment per module → 3 total**
